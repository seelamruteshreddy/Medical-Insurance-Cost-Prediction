 * Medical-Insurance-Cost-Prediction
Machine Learning project to predict medical insurance charges using age, BMI, children, smoking status, gender, and region

 Project Overview
This project predicts medical insurance charges using machine learning.

 Dataset Features
- Age
- Gender
- BMI
- Children
- Smoker
- Region

 Model Used
- Linear Regression

 Libraries
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn

 Results
The model predicts insurance charges and is evaluated using R² Score, MAE, and MSE.
